# CLAUDE.md - vijil-domed-travel-agent

**Company context:** @../CLAUDE.md

Demo travel agent with Dome protection layer integrated.

## Purpose

This variant of the travel agent demonstrates:
- Dome guardrails in action
- Real-time input/output filtering
- Detection and blocking of malicious requests

## Difference from vijil-travel-agent

- `vijil-travel-agent` - Unprotected agent for baseline evaluation
- `vijil-domed-travel-agent` - Same agent with Dome protection active

This allows A/B comparison of protected vs unprotected agent behavior.
